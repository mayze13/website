A pipeline that processes PDF documents and generates high-quality question-and-answer pairs from their content. Particularly useful for creating training datasets, study materials, or knowledge-base evaluation sets.

**Key features:**
- PDF parsing and chunking with smart context preservation
- Anonymisation of sensitive entities before processing
- Chain of Density refinement for progressively richer Q&A pairs
- Configurable output format (JSON, CSV)

**Stack:** Python · LangChain · pdfplumber · OpenAI API

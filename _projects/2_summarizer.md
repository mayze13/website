An automated summarisation pipeline that retrieves video transcription data from AWS and distils it into structured outputs: key insights, main objectives, and recurring themes. Designed to save teams hours of manual review time.

**Key features:**
- AWS integration for transcription retrieval
- LLM-powered extraction of insights, objectives, and themes
- Structured output format for downstream use
- Handles long-form transcripts reliably

**Stack:** Python · LangChain · AWS · OpenAI API

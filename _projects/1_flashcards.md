A generative AI proof-of-concept built for students learning to become coaches. The tool takes a knowledge base and automatically produces personalised, targeted flashcards using Retrieval-Augmented Generation (RAG) and Chain of Density summarisation.

**Key features:**
- Document ingestion and vectorisation via LangChain
- RAG pipeline to ground flashcard generation in source material
- Chain of Density to produce progressively denser, more informative cards
- Clean Streamlit interface for immediate usability

**Stack:** Python · LangChain · Streamlit · OpenAI API · FAISS
